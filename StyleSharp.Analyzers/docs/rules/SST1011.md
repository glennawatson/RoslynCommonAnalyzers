# SST1011 — Closing square brackets should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

A closing square bracket `]` is preceded by a space.

## Rule description

```csharp
// 👎 reported
var first = items[0 ];

// 👍
var first = items[0];
```

Attribute brackets are governed by [SST1017](SST1017.md).

The StyleSharp counterpart of StyleCop's `SA1011`.

## How to fix violations

The code fix removes the space before the `]`.
