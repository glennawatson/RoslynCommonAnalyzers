# SST1135 — Using directives should be qualified

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes (fully qualify the name) |

## Cause

A using directive names a namespace or type in a context-relative form rather than fully qualified.

## Rule description

A fully qualified using does not depend on the enclosing namespace, so it cannot change meaning when
the file is moved or another namespace is introduced.

```csharp
namespace System.Threading
{
    // 👎 reported — relative to the enclosing namespace
    using Tasks;

    // 👍
    using System.Threading.Tasks;
}
```

The StyleSharp counterpart of StyleCop's `SA1135`.

## How to fix violations

The code fix rewrites the directive with the fully qualified name.
