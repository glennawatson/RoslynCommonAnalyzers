# SST1511 — Do/while footer should not be preceded by a blank line

| Item | Value |
| --- | --- |
| Category | Layout |
| Severity | Warning |
| Code fix | Yes (remove blank line) |

## Cause

A blank line precedes the `while` footer of a do/while loop.

## Rule description

The `while` footer follows the loop body directly.

```csharp
// 👎 reported
do
{
}

while (condition);

// 👍 footer follows the body
do
{
}
while (condition);
```

The StyleSharp counterpart of StyleCop's `SA1511`.

## How to fix violations

The code fix removes the blank line(s) before the `while` footer.
