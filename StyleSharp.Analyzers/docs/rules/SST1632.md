# SST1632 — Documentation text should meet a minimum length

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning (disabled by default) |
| Code fix | No |

## Cause

A documentation summary is too short to be meaningful.

## Rule description

Off by default, matching StyleCop. Enable it to flag summaries that are too short to
describe the element.

The StyleSharp counterpart of StyleCop's `SA1632`.

## How to fix violations

Write a fuller summary.
