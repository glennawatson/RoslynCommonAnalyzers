# PSH1009 — Bound variable-length stackalloc with a constant guard

**Category:** Allocations
**Severity:** Warning
**Code fix:** None (the guard shape is a design decision)

A `stackalloc` whose length comes from data can blow the stack on adversarial
or unexpected input. The resilient shape tests the length against a constant
first and takes a heap or pooled buffer above the threshold, keeping the
stack fast path for small sizes.

The rule is false-positive-averse: anything that looks bounded is trusted. A
constant or `const`/`static readonly` field length, a `Math.Min`/`Math.Clamp`
length, a surrounding conditional whose condition compares against a
constant, and an enclosing `if` guard (including relational patterns such as
`length is <= 512`) all keep the expression clean.

## Example

```csharp
public int M(int length)
{
    Span<byte> buffer = stackalloc byte[length]; // PSH1009: no constant bound in sight

    // Preferred shape:
    Span<byte> bounded = length <= 512 ? stackalloc byte[length] : new byte[length];
    return bounded.Length;
}
```
