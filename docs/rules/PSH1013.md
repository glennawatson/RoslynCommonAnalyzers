# PSH1013 — Expose constant UTF-8 data as a ReadOnlySpan<byte> property

**Category:** Allocations
**Severity:** Warning
**Code fix:** Converts the field to an expression-bodied span property (supports Fix All)

A `private static readonly byte[]` built from a u8 literal — directly through
`"..."u8.ToArray()` or a spread like `[.. "..."u8]` — allocates a mutable
array at startup and pays a field load on every read. A
`static ReadOnlySpan<byte>` property returning the literal compiles to a
direct pointer into the assembly's data section: no allocation, no
indirection, and the data can never be mutated.

Reported only when every use already reads the field like a span — `Length`,
element reads, `foreach`, and arguments bound to `ReadOnlySpan<byte>`
parameters — so the rewritten property compiles everywhere the field was
used. Partial types are skipped.

## Example

```csharp
private static readonly byte[] Prefix = "v1:"u8.ToArray(); // PSH1013
private static ReadOnlySpan<byte> Prefix => "v1:"u8;       // fixed
```
