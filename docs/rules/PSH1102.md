# PSH1102 - Filter LINQ values by type once

| Item | Value |
| --- | --- |
| Package | PerformanceSharp.Analyzers |
| Category | Collections |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

> Moved from StyleSharp.Analyzers, where it shipped as SST2230.

A LINQ type check followed by `Cast<T>` makes two operators cooperate to do one type-filtering job.
`OfType<T>` expresses the same local pipeline shape with one operator.

## Example

```csharp
return values.Where(value => value is string).Cast<string>();
```

Use:

```csharp
return values.OfType<string>();
```

## Notes

The analyzer only reports simple `value is T` predicates followed by `Cast<T>()`, and verifies the calls
resolve to in-memory LINQ before reporting.
