# PSH1205 — Remove interpolation that does no work

| Item | Value |
| --- | --- |
| Package | PerformanceSharp.Analyzers |
| Category | Strings |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

An interpolated string either wraps a single string value in a bare hole, or
contains no holes at all.

```csharp
// 👎 reported
string copy = $"{name}";
string text = $"hello";

// 👍
string copy = name;
string text = "hello";
```

## Rule description

An interpolated string that does no interpolation work still runs the
interpolation machinery and allocates a fresh string. (the rule-inspired.) Two
shapes are reported when the interpolated string's **converted type is
`string`**:

- **A single bare string hole** — exactly one hole, no alignment, no format
  clause, whose expression is already a `string`. The value itself is the result.
- **No holes at all** — only literal text (possibly empty). A plain string
  literal is free.

The following are **not** reported:

- an alignment (`$"{s,10}"`) or format clause (`$"{s:F2}"`) — the interpolation
  is doing padding or formatting work;
- a non-string hole (`$"{obj}"`) — interpolation supplies `ToString`/formatting
  semantics the raw value does not have;
- mixed text and holes (`$"a{x}"`) — the interpolation concatenates;
- a converted type other than `string` — `FormattableString`, `IFormattable`, or
  a custom interpolated-string handler — where the interpolated form carries
  structure a plain value cannot.

### Null caveat for the single-hole shape

`$"{s}"` renders a null `s` as `""`, while the raw value stays null. If the use
site relies on that normalization, interpolation is an expensive way to write
`s ?? ""` — prefer the explicit coalesce over keeping the interpolation.

## How to fix violations

Use the value or the plain literal directly. The provided code fix replaces the
single-hole shape with its expression (parenthesized when the expression is not
a primary expression, for example a conditional or a concatenation) and the
text-only shape with a regular literal of equal value, unescaping `{{`/`}}`.
