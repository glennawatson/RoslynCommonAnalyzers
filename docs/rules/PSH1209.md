# PSH1209 — Build transformed strings with string.Create

**Category:** Strings
**Severity:** Warning
**Code fix:** None (the rewrite moves the loop into a builder lambda)

The copy-mutate-rebuild shape — `ToCharArray()`, in-place element writes,
then `new string(chars)` — allocates two throwaway buffers to produce one
result. `string.Create` allocates the final string once and exposes its
buffer as a writable span, so the transformation writes directly into the
result.

All three parts are matched in one pass over the enclosing block before any
binding. Suggested only where `string.Create` and `SpanAction` exist.

## Example

```csharp
var chars = input.ToCharArray();      // PSH1209
chars[0] = char.ToUpperInvariant(chars[0]);
return new string(chars);

return string.Create(input.Length, input, static (buffer, source) =>
{
    source.AsSpan().CopyTo(buffer);
    buffer[0] = char.ToUpperInvariant(buffer[0]);
});
```
