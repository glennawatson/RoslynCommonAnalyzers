# PSH1210 — Compare UTF-8 bytes without decoding them

**Category:** Strings
**Severity:** Warning
**Code fix:** Rewrites the comparison to `SequenceEqual` against a u8 literal (supports Fix All)

Decoding bytes with `Encoding.UTF8.GetString` only to compare the result
against a constant allocates a string and throws it away.
`SequenceEqual` against a u8 literal (C# 11+) compares the raw bytes and
allocates nothing. Array sources gain an `AsSpan()` so the span extension
binds on every language version; inequalities gain a leading negation.

Constants containing U+FFFD are skipped — an invalid byte sequence decodes
to that character, so the string comparison and the byte comparison could
disagree.

## Example

```csharp
if (Encoding.UTF8.GetString(payload) == "ok")   // PSH1210
if (payload.AsSpan().SequenceEqual("ok"u8))     // fixed
```
