# PSH1212 — Slice with AsSpan when the call accepts a span

**Category:** Strings
**Severity:** Warning
**Code fix:** Renames the call to `AsSpan` (supports Fix All)

`Substring` allocates a new string for the slice. When the invoked method
has an overload accepting `ReadOnlySpan<char>` in the same position,
`AsSpan` produces the slice with no allocation — its overloads mirror
Substring's start and length arguments, so the rename carries the arguments
through unchanged.

Reported only when the span overload exists (identical apart from that one
parameter) and `AsSpan` resolves at the call site, so the fix always
compiles.

## Example

```csharp
Use(value.Substring(1));   // PSH1212 — Use(ReadOnlySpan<char>) exists
Use(value.AsSpan(1));      // fixed
```
