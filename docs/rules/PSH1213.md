# PSH1213 — Probe repeated character sets through SearchValues

**Category:** Strings
**Severity:** Warning
**Code fix:** None (the rewrite hoists a static field the type must adopt)

`IndexOfAny`, `ContainsAny`, and their variants rescan an inline constant
set linearly on every call, and the inline array can allocate each time. A
`static readonly SearchValues<char>` (.NET 8+) precomputes the membership
table once, and the overloads that take it probe in constant time per
character:

```csharp
private static readonly SearchValues<char> Separators = SearchValues.Create(";,|");

public int M(ReadOnlySpan<char> text) => text.IndexOfAny(Separators);
```

Reported for inline constant char arrays and collection expressions passed
to the string and span search APIs. Suggested only where `SearchValues`
exists.

## Example

```csharp
value.IndexOfAny(new[] { ';', ',', '|' });  // PSH1213
text.ContainsAny([';', ',']);               // PSH1213
```
