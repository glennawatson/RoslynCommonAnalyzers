# PSH1216 — Ask for equality, not ordering

**Category:** Strings
**Severity:** Warning
**Code fix:** Rewrites the comparison to `string.Equals` with the matching `StringComparison` (supports Fix All)

`string.Compare`, `string.CompareOrdinal`, and `CompareTo` compute full ordering
information — less-than, equal, greater-than — that a `== 0` or `!= 0` test
immediately throws away. `string.Equals` answers the question that was actually
asked and can bail out early, for example on a length mismatch under ordinal
rules, instead of walking both strings far enough to rank them.

The fix preserves the culture semantics the call already had. The subtle part is
that the two-argument `string.Compare` and instance `CompareTo` compare using the
**current culture**, not ordinal, so a naive rewrite to `string.Equals(a, b)`
(which is ordinal) would silently change behavior. Each shape maps to the
comparison it was already performing:

| Original | Replacement |
| --- | --- |
| `string.Compare(a, b) == 0` | `string.Equals(a, b, StringComparison.CurrentCulture)` |
| `string.Compare(a, b, comparison) == 0` | `string.Equals(a, b, comparison)` |
| `string.Compare(a, b, true) == 0` | `string.Equals(a, b, StringComparison.CurrentCultureIgnoreCase)` |
| `string.Compare(a, b, false) == 0` | `string.Equals(a, b, StringComparison.CurrentCulture)` |
| `string.CompareOrdinal(a, b) == 0` | `string.Equals(a, b, StringComparison.Ordinal)` |
| `a.CompareTo(b) == 0` | `string.Equals(a, b, StringComparison.CurrentCulture)` |

`!= 0` forms become `!string.Equals(...)`, and the literal `0` may sit on either
side of the operator. Real ordering tests (`string.Compare(a, b) > 0`) are not
reported. The three-argument `ignoreCase` shape is only reported when the flag is
a literal `true`/`false` — with a runtime flag no single `StringComparison` value
is a safe replacement, so that call stays silent.

## Example

```csharp
if (string.Compare(name, other) == 0) { }                              // PSH1216
if (string.Equals(name, other, StringComparison.CurrentCulture)) { }   // fixed
```
