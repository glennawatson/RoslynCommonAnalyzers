# PSH1305 — Enumerate a ConcurrentDictionary directly instead of a Keys or Values snapshot

**Category:** Concurrency
**Severity:** Warning
**Code fix:** Deconstructs the pair (supports Fix All)

`ConcurrentDictionary`'s `Keys` and `Values` properties lock every bucket
and copy the whole collection into a fresh list on each access. Enumerating
the dictionary itself is lock-free and allocates only an enumerator.

Only `foreach` over `.Keys`/`.Values` is reported. The fix rewrites the loop
into a pair deconstruction so the loop variable keeps its name and the body
is untouched; it is offered where `KeyValuePair` exposes `Deconstruct`
(.NET Core 2.0+ / netstandard2.1+).

## Example

```csharp
foreach (var key in map.Keys)   // PSH1305
foreach (var (key, _) in map)   // fixed

foreach (var value in map.Values) // PSH1305
foreach (var (_, value) in map)   // fixed
```
