# PSH1406 - Ask Regex for the answer directly

| Item | Value |
| --- | --- |
| Package | PerformanceSharp.Analyzers |
| Category | ApiSelection |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

Code materializes regex matches only to read whether a match exists or how many
matches were found.

```csharp
if (regex.Match(input).Success) // PSH1406
{
}

var count = regex.Matches(input).Count; // PSH1406
```

## Rule description

`Regex.Match(...).Success` creates a `Match` object when the caller only needs a
boolean answer. `Regex.Matches(...).Count` creates a match collection where
newer frameworks can answer the count directly.

The analyzer handles static and instance regex calls. Count rewrites are gated on
the direct `Regex.Count` API being available.

## How to fix violations

Use `IsMatch` or `Count`.

```csharp
if (regex.IsMatch(input))
{
}

var count = regex.Count(input);
```

## When to suppress

Suppress when the match object or collection is already needed for more than the
single boolean or count value.
