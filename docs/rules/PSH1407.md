# PSH1407 - Query dictionaries directly

| Item | Value |
| --- | --- |
| Package | PerformanceSharp.Analyzers |
| Category | ApiSelection |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

Code calls `Contains` on a dictionary `Keys` view.

```csharp
if (map.Keys.Contains(key)) // PSH1407
{
}
```

## Rule description

`dictionary.Keys.Contains(key)` goes through the key collection view and may add
extra work before the dictionary can answer the lookup. `ContainsKey` asks the
dictionary directly and performs a single key probe.

The rule reports only when the receiver exposes an accessible `ContainsKey`
method for the key type.

## How to fix violations

Use `ContainsKey`.

```csharp
if (map.ContainsKey(key))
{
}
```

## When to suppress

Suppress only when the expression is intentionally operating on a non-dictionary
key collection abstraction.
