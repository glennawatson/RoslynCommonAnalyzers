# PSH1408 — Measure elapsed time with Stopwatch timestamps

**Category:** ApiSelection
**Severity:** Warning
**Code fix:** None (the rewrite spans multiple statements)

A `Stopwatch` allocated only to read elapsed time can be replaced by
capturing `Stopwatch.GetTimestamp()` into a `long` and asking
`Stopwatch.GetElapsedTime(...)` for the difference (.NET 7+) — same
precision, no allocation.

The whole rule is gated on `GetElapsedTime` existing in the compilation. A
`Stopwatch.StartNew()` local is reported when its only uses are `Elapsed`,
`ElapsedMilliseconds`, or `ElapsedTicks` reads and `Stop()` calls. Anything
else — `Restart`, `Reset`, `IsRunning`, escaping as an argument or return
value — keeps the declaration clean.

## Example

```csharp
var stopwatch = Stopwatch.StartNew(); // PSH1408
DoWork();
Log(stopwatch.Elapsed);

// Preferred shape:
var start = Stopwatch.GetTimestamp();
DoWork();
Log(Stopwatch.GetElapsedTime(start));
```
