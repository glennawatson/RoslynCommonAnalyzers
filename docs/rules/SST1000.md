# SST1000 — Keywords should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (add space) |

## Cause

A control-flow keyword (`if`, `while`, `for`, `foreach`, `switch`, `lock`, `using`,
`fixed`, `catch`) is immediately followed by its opening parenthesis.

## Rule description

```csharp
// 👎 reported
if(enabled)
{
}

// 👍
if (enabled)
{
}
```

The StyleSharp counterpart of the analyzer's `the rule` (the parenthesised-keyword case).

## How to fix violations

The code fix inserts a space after the keyword.
