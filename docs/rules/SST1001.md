# SST1001 — Commas should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes |

## Cause

A comma is preceded by whitespace, or is not followed by a space.

## Rule description

A comma is followed by a single space (or a line break) and is never preceded by
whitespace.

```csharp
// 👎 reported
Combine(a ,b);

// 👍 spaced correctly
Combine(a, b);
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix removes whitespace before the comma or inserts a space after it.
