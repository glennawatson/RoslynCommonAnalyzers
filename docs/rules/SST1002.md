# SST1002 — Semicolons should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes |

## Cause

A semicolon is preceded by whitespace, or is not followed by a space when more code
follows it on the same line.

## Rule description

A semicolon is never preceded by whitespace, and is followed by a space (or line break)
when it is not the last token before a closing parenthesis (as in `for (;;)`).

```csharp
// 👎 reported
Work() ;

// 👍 spaced correctly
Work();
```

## How to fix violations

The code fix removes whitespace before the semicolon or inserts a space after it.
