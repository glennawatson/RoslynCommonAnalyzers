# SST1003 — Symbols should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (add space) |

## Cause

A binary or assignment operator is missing a space on one or both sides.

## Rule description

Binary and assignment operators (`+`, `-`, `*`, `/`, `==`, `&&`, `=`, `+=`, `??`, …) have
a single space on each side. Unary operators are governed by
[SST1021](SST1021.md)/[SST1022](SST1022.md), and generic brackets by
[SST1014](SST1014.md)/[SST1015](SST1015.md).

```csharp
// 👎 reported
var total = a+b;

// 👍
var total = a + b;
```

The StyleSharp counterpart of StyleCop's `SA1003` (binary-operator case).

## How to fix violations

The code fix inserts the missing space.
