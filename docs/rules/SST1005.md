# SST1005 — Single-line comments should begin with a single space

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (add space) |

## Cause

A single-line comment's text starts immediately after `//` with no space.

## Rule description

A single-line comment begins with `// ` so the text is easy to read. Comments that start
with extra slashes (`////`, used to comment out code) are exempt.

```csharp
// 👎 reported
//calculates the total

// 👍 single leading space
// calculates the total
```

## How to fix violations

The code fix inserts a space after the `//`.
