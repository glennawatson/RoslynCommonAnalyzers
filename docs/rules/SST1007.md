# SST1007 — Operator keyword should be followed by a space

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (add space) |

## Cause

The `operator` keyword is immediately followed by the operator symbol or target type.

## Rule description

```csharp
// 👎 reported
public static C operator+(C a, C b) => a;

// 👍
public static C operator +(C a, C b) => a;
```

The StyleSharp counterpart of StyleCop's `SA1007`.

## How to fix violations

The code fix inserts a space after the `operator` keyword.
