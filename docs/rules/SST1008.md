# SST1008 — Opening parenthesis should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

An opening parenthesis is followed by a space on the same line.

## Rule description

```csharp
// 👎 reported
Configure( value);

// 👍
Configure(value);
```

Whether the `(` should be *preceded* by a space is keyword-dependent and is governed by
[SST1000](SST1000.md) for control-flow keywords.

The StyleSharp counterpart of the analyzer's `the rule` (the no-space-after case).

## How to fix violations

The code fix removes the space after the `(`.
