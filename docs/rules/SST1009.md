# SST1009 — Closing parenthesis should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

A closing parenthesis is preceded by a space.

## Rule description

```csharp
// 👎 reported
Configure(value );

// 👍
Configure(value);
```

## How to fix violations

The code fix removes the space before the `)`.
