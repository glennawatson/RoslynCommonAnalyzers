# SST1010 — Opening square brackets should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning (disabled by default) |
| Code fix | Yes (remove space) |

## Cause

An opening square bracket (element access or array) has an adjacent space.

## Rule description

This rule is **disabled by default** because it conflicts with the C# 12 collection-
expression style (`x = [1, 2]`, where a space before `[` is wanted). Enable it in
`.editorconfig` if your project does not use collection expressions.

```csharp
// 👎 reported
var first = items [0];

// 👍
var first = items[0];
```

Attribute brackets are governed by [SST1016](SST1016.md).

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix removes the space adjacent to the `[`.
