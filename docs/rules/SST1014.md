# SST1014 — Opening generic brackets should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

The `<` of a generic argument or parameter list is preceded or followed by a space.

## Rule description

```csharp
// 👎 reported
List <int> values;

// 👍
List<int> values;
```

## How to fix violations

The code fix removes the offending space.
