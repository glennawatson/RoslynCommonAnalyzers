# SST1015 — Closing generic brackets should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

The `>` of a generic argument or parameter list is preceded by a space.

## Rule description

```csharp
// 👎 reported
List<int > values;

// 👍
List<int> values;
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix removes the space before the `>`.
