# SST1016 — Opening attribute brackets should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

The `[` of an attribute list is followed by a space.

## Rule description

```csharp
// 👎 reported
[ Obsolete]

// 👍
[Obsolete]
```

## How to fix violations

The code fix removes the space after the `[`.
