# SST1017 — Closing attribute brackets should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

The `]` of an attribute list is preceded by a space.

## Rule description

```csharp
// 👎 reported
[Obsolete ]

// 👍
[Obsolete]
```

The StyleSharp counterpart of StyleCop's `SA1017`.

## How to fix violations

The code fix removes the space before the `]`.
