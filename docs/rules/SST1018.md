# SST1018 — Nullable type symbols should not be preceded by a space

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

The `?` of a nullable type is preceded by a space.

## Rule description

```csharp
// 👎 reported
int ? value;

// 👍
int? value;
```

## How to fix violations

The code fix removes the space before the `?`.
