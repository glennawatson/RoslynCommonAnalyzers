# SST1019 — Member access symbols should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

A member access `.` is preceded or followed by a space.

## Rule description

A member access `.` has no adjacent whitespace. A leading dot on its own line (fluent
style) is allowed.

```csharp
// 👎 reported
value .ToString();

// 👍
value.ToString();
```

## How to fix violations

The code fix removes the offending space.
