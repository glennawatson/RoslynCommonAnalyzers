# SST1020 — Increment and decrement symbols should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

A `++` or `--` operator is separated from its operand by a space.

## Rule description

```csharp
// 👎 reported
count ++;
-- count;

// 👍
count++;
--count;
```

## How to fix violations

The code fix removes the space between the operator and its operand.
