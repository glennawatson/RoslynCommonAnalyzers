# SST1021 — Negative signs should be spaced correctly

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

A unary negative sign is followed by a space.

## Rule description

```csharp
// 👎 reported
var n = - value;

// 👍
var n = -value;
```

The StyleSharp counterpart of StyleCop's `SA1021`.

## How to fix violations

The code fix removes the space after the unary `-`.
