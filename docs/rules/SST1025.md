# SST1025 — Code should not contain multiple whitespace characters in a row

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (collapse to a single space) |

## Cause

Two or more whitespace characters appear in a row between tokens on a line.

## Rule description

A single space separates code on a line; runs of whitespace used for alignment are
collapsed. Leading indentation is not affected.

```csharp
// 👎 reported
var x =  1;

// 👍 single space
var x = 1;
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix replaces the run of whitespace with a single space.
