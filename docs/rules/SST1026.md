# SST1026 — Implicit array creation should not contain a space before the bracket

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove space) |

## Cause

A space appears between `new` (or `stackalloc`) and the `[` of an implicit array creation.

## Rule description

```csharp
// 👎 reported
var values = new [] { 1, 2, 3 };

// 👍
var values = new[] { 1, 2, 3 };
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix removes the space before the `[`.
