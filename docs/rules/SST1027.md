# SST1027 — Use spaces rather than tabs

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (replace tabs with spaces) |

## Cause

Whitespace contains a tab character.

## Rule description

Whitespace is made of spaces rather than tabs, matching the repository's `indent_style =
space`. Projects that standardise on tabs can disable this rule.

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix replaces each tab with spaces.
