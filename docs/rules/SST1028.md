# SST1028 — Code should not contain trailing whitespace

| Item | Value |
| --- | --- |
| Category | Spacing |
| Severity | Warning |
| Code fix | Yes (remove trailing whitespace) |

## Cause

A line ends with one or more whitespace characters.

## Rule description

Lines end at their last non-whitespace character, with no trailing spaces or tabs.

## How to fix violations

The code fix removes the trailing whitespace.
