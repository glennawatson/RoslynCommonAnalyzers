# SST1116 - Prefer the shortest symbol-equivalent name

Qualified type and namespace names should be shortened when the shorter spelling resolves to the
same symbol. The analyzer first looks for a qualified-name shape and then asks Roslyn whether the
short name binds to the same type or namespace, so it avoids reporting names that would become
ambiguous.

## Example

```csharp
using System.Text;

public int Count(System.Text.StringBuilder builder) => builder.Length;
```

Use:

```csharp
using System.Text;

public int Count(StringBuilder builder) => builder.Length;
```

## Notes

This rule intentionally skips namespace declarations and using directives. Its hot path is a
syntax-kind filter; semantic speculative binding only runs after a candidate qualified name is
found.
