# SST1117 - Drop redundant this qualification

`this.` should be left out when the member name alone resolves to the same member. The rule keeps
the code visually lighter while still preserving explicit qualification in shadowing cases.

## Example

```csharp
public sealed class Counter
{
    private readonly int _value;

    public int Value => this._value;
}
```

Use:

```csharp
public sealed class Counter
{
    private readonly int _value;

    public int Value => _value;
}
```

## Notes

The analyzer verifies the unqualified name with speculative binding before reporting. If a local or
parameter would capture the shortened name, no diagnostic is produced.
