# SST1125 — Use shorthand for nullable types

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes (rewrite as `T?`) |

## Cause

A nullable value type is written in long form as `Nullable<T>` or `System.Nullable<T>`.

## Rule description

The `T?` shorthand is shorter and the idiomatic way to spell a nullable value type.

```csharp
// 👎 reported
private Nullable<int> count;

// 👍
private int? count;
```

The semantic model confirms the type is `System.Nullable<T>`, so a user-defined type named
`Nullable` is never flagged.

The StyleSharp counterpart of StyleCop's `SA1125`.

## How to fix violations

The code fix rewrites the type as the `T?` shorthand.
