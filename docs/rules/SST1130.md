# SST1130 — Use lambda syntax

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | No |

## Cause

An inline callback is written with the older anonymous-method `delegate` syntax.

## Rule description

Lambda expressions are shorter, support expression bodies, and are the idiomatic way to write an
inline callback in modern C#.

```csharp
// 👎 reported
Action<int> log = delegate (int x) { Console.WriteLine(x); };

// 👍
Action<int> log = x => Console.WriteLine(x);
```

The StyleSharp counterpart of StyleCop's `SA1130`.

## How to fix violations

Rewrite the anonymous delegate as a lambda expression, replacing `delegate (args)` with
`(args) =>`.
