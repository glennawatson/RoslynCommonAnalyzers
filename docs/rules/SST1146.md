# SST1146 — An if statement should start on a new line

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

An independent `if` follows a closing brace on the same line.

```csharp
if (first) { } if (second) { }
```

## Rule description

Starting the second statement on a new line makes it clear that it is independent rather
than an `else if`. The rule does not report normal `else if` clauses.

## How to fix violations

Move the `if` statement to the next line. The provided code fix does this automatically.
