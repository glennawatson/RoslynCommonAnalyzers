# SST1151 - Method declaration parameters should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

A method parameter list uses a jagged multi-line layout.

```csharp
void Save(string id, string name,
    DateTime created) { } // SST1151
```

## Rule description

Multi-line method parameter lists should put each parameter on its own line so
the signature remains stable and easy to scan.

## How to fix violations

Use one parameter per line, or collapse the list back to one line when it is
short enough.

```csharp
void Save(
    string id,
    string name,
    DateTime created) { }
```

## When to suppress

Prefer an `.editorconfig` scope for generated or intentionally formatted code.
