# SST1156 - Element access arguments should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

An element access argument list wraps unevenly.

```csharp
var value = grid[row, column,
    layer]; // SST1156
```

## Rule description

Multi-line element access should put each index argument on its own line so
multi-dimensional access remains readable.

## How to fix violations

Place each index argument on its own line.

```csharp
var value = grid[
    row,
    column,
    layer];
```

## When to suppress

Use scoped configuration for generated code or source that must keep another
layout.
