# SST1157 - Attribute arguments should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

An attribute argument list is split across lines with multiple arguments on the
same line.

```csharp
[Example("id", "name",
    Enabled = true)] // SST1157
public sealed class Widget;
```

## Rule description

Wrapped attribute arguments should be listed one per line so metadata remains
easy to review.

## How to fix violations

Put each positional or named argument on its own line.

```csharp
[Example(
    "id",
    "name",
    Enabled = true)]
public sealed class Widget;
```

## When to suppress

Prefer scoped configuration for generated attribute lists.
