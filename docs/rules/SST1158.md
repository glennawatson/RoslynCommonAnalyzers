# SST1158 - Anonymous method parameters should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

An anonymous method parameter list wraps inconsistently.

```csharp
Action<string, string, CancellationToken> action = delegate(string id, string name,
    CancellationToken cancellationToken) { }; // SST1158
```

## Rule description

When anonymous method parameters span multiple lines, each parameter should be on
its own line.

## How to fix violations

Use one parameter per line.

```csharp
Action<string, string, CancellationToken> action = delegate(
    string id,
    string name,
    CancellationToken cancellationToken) { };
```

## When to suppress

Use scoped configuration for generated or externally formatted code.
