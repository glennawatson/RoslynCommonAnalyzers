# SST1159 - Parenthesized lambda parameters should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

A parenthesized lambda parameter list uses a jagged multi-line layout.

```csharp
var action = (string id, string name,
    CancellationToken cancellationToken) => Save(id, name, cancellationToken); // SST1159
```

## Rule description

Wrapped lambda parameter lists should use one parameter per line. This keeps
delegate-shaped code readable without changing the lambda body.

## How to fix violations

Put each parameter on its own line.

```csharp
var action = (
    string id,
    string name,
    CancellationToken cancellationToken) => Save(id, name, cancellationToken);
```

## When to suppress

Prefer scoped configuration for generated lambda-heavy code.
