# SST1162 - Struct primary-constructor parameters should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

A struct primary-constructor parameter list wraps unevenly.

```csharp
public readonly struct Range(int Start, int End,
    int Step); // SST1162
```

## Rule description

Wrapped struct primary-constructor parameters should put every parameter on its
own line.

## How to fix violations

Use one parameter per line.

```csharp
public readonly struct Range(
    int Start,
    int End,
    int Step);
```

## When to suppress

Use scoped configuration for generated structs.
