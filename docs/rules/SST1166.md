# SST1166 - Local function parameters should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

A local function parameter list wraps with some parameters sharing a line.

```csharp
void Build(string id, string name,
    DateTime created) { } // SST1166
```

## Rule description

Wrapped local function parameters should follow the same one-per-line layout as
regular method parameters.

## How to fix violations

Use one parameter per line.

```csharp
void Build(
    string id,
    string name,
    DateTime created) { }
```

## When to suppress

Prefer scoped configuration for generated code.
