# SST1167 - Operator declaration parameters should be on unique lines

| Item | Value |
| --- | --- |
| Package | StyleSharp.Analyzers |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

An operator declaration parameter list wraps unevenly.

```csharp
public static Vector operator +(Vector left,
    Vector right) => left.Add(right); // SST1167
```

## Rule description

Operator declarations should keep multi-line parameter lists regular, with one
parameter per line.

## How to fix violations

Place each parameter on its own line.

```csharp
public static Vector operator +(
    Vector left,
    Vector right) => left.Add(right);
```

## When to suppress

Prefer scoped configuration for generated operators.
