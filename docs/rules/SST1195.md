# SST1195 - Write null fallback with ??

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |
| Analyzer | `LanguageStyleAnalyzer` |

## Cause

A conditional expression checks a value for `null`, returns a fallback when it is null, and returns
the original value otherwise.

```csharp
return value == null ? fallback : value;
```

## Rule description

The null-coalescing operator expresses this fallback directly. The analyzer only reports local,
parameter, or field reads so the suggested form does not duplicate a property or method call.

## How to fix violations

Write the fallback with `??`.
