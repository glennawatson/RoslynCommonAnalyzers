# SST1196 - Write null-guarded access with ?.

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |
| Analyzer | `LanguageStyleAnalyzer` |

## Cause

A conditional expression checks a receiver for `null`, returns `null` when it is null, and reads a
member from the receiver otherwise.

```csharp
return person == null ? null : person.Name;
```

## Rule description

The null-propagation operator expresses this guard directly. The analyzer reports only stable
local, parameter, or field receivers.

## How to fix violations

Write the guarded access with `?.`.
