# SST1197 - Collapse return-only branch into one return

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |
| Analyzer | `LanguageStyleAnalyzer` |

## Cause

An `if` statement returns one value and the following statement returns the alternative value.

```csharp
if (flag)
{
    return 1;
}

return 2;
```

## Rule description

When an `if` only chooses between two return values, a single return statement with a conditional
expression keeps the value selection compact.

## How to fix violations

Collapse the branch into one return statement with a conditional expression.
