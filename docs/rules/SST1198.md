# SST1198 - Collapse assignment-only branch into one assignment

| Item | Value |
| --- | --- |
| Category | Readability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |
| Analyzer | `LanguageStyleAnalyzer` |

## Cause

Both branches of an `if`/`else` statement assign to the same local, parameter, or field.

```csharp
if (flag)
{
    value = 1;
}
else
{
    value = 2;
}
```

## Rule description

When both branches only assign the same target, a conditional expression makes the value choice
clear at the assignment site.

## How to fix violations

Collapse the branch into one assignment whose value is a conditional expression.
