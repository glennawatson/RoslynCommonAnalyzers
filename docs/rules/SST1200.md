# SST1200 — Using directives should be placed outside the namespace

| Item | Value |
| --- | --- |
| Category | Ordering |
| Severity | Warning |
| Code fix | No |

## Cause

A using directive is placed inside a namespace declaration.

## Rule description

Using directives are placed at the top of the file, outside the namespace declaration,
matching the .NET runtime convention.

```csharp
// 👎 reported
namespace Example
{
    using System;
}

// 👍 outside the namespace
using System;

namespace Example
{
}
```

The StyleSharp counterpart of StyleCop's `SA1200`, adapted to the .NET runtime's
outside-namespace convention.

## How to fix violations

Move the using directives above the namespace declaration.
