# SST1205 — Partial elements should declare an access modifier

| Item | Value |
| --- | --- |
| Category | Ordering |
| Severity | Warning |
| Code fix | Yes (add access modifier) |

## Cause

A partial type or partial method does not declare an access modifier.

## Rule description

Partial declarations declare their accessibility explicitly so every part agrees on the
intended access. The implicit default is `internal` for a top-level type and `private`
otherwise.

```csharp
// 👎 reported
partial class Widget
{
}

// 👍 explicit access
internal partial class Widget
{
}
```

The StyleSharp counterpart of StyleCop's `SA1205`.

## How to fix violations

The code fix adds the implicit access modifier.
