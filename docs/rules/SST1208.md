# SST1208 — System using directives should appear before other usings

| Item | Value |
| --- | --- |
| Category | Ordering |
| Severity | Warning |
| Code fix | Yes (sort using directives) |

## Cause

A using directive in the `System` namespace appears after a non-System using directive.

## Rule description

Regular using directives in the `System` namespace (and its children) appear before
other regular using directives.

```csharp
// 👎 reported
using Contoso;
using System;

// 👍 System first
using System;
using Contoso;
```

The StyleSharp counterpart of StyleCop's `SA1208`.

## How to fix violations

The code fix sorts the using directives into the canonical order.
