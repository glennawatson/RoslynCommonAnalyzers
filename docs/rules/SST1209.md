# SST1209 — Using alias directives should appear after other usings

| Item | Value |
| --- | --- |
| Category | Ordering |
| Severity | Warning |
| Code fix | Yes (sort using directives) |

## Cause

A using alias directive (`using X = ...;`) appears before a regular or static using
directive.

## Rule description

Using alias directives appear after all regular and `using static` directives.

```csharp
// 👎 reported
using Con = System.Console;
using System.Text;

// 👍 aliases last
using System.Text;
using Con = System.Console;
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix sorts the using directives into the canonical order.
