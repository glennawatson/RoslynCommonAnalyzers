# SST1211 — Using alias directives should be ordered alphabetically by alias name

| Item | Value |
| --- | --- |
| Category | Ordering |
| Severity | Warning |
| Code fix | Yes (sort using directives) |

## Cause

Using alias directives are not ordered alphabetically by alias name.

## Rule description

Using alias directives are ordered alphabetically by the alias name.

```csharp
// 👎 reported
using Y = System.Text.StringBuilder;
using X = System.Console;

// 👍 alphabetical by alias
using X = System.Console;
using Y = System.Text.StringBuilder;
```

## How to fix violations

The code fix sorts the using directives into the canonical order.
