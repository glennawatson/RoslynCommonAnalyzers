# SST1216 — Using static directives should appear after regular usings and before aliases

| Item | Value |
| --- | --- |
| Category | Ordering |
| Severity | Warning |
| Code fix | Yes (sort using directives) |

## Cause

A `using static` directive is not placed between the regular and alias using directives.

## Rule description

`using static` directives appear after all regular using directives and before any using
alias directives.

```csharp
// 👎 reported
using static System.Math;
using System.Text;

// 👍 static after regular
using System.Text;
using static System.Math;
```

## How to fix violations

The code fix sorts the using directives into the canonical order.
