# SST1217 — Using static directives should be ordered alphabetically

| Item | Value |
| --- | --- |
| Category | Ordering |
| Severity | Warning |
| Code fix | Yes (sort using directives) |

## Cause

`using static` directives are not ordered alphabetically.

## Rule description

`using static` directives are ordered alphabetically by the type name.

```csharp
// 👎 reported
using static System.Math;
using static System.Console;

// 👍 alphabetical
using static System.Console;
using static System.Math;
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

The code fix sorts the using directives into the canonical order.
