# SST1300 — Element names should be PascalCase

| Item | Value |
| --- | --- |
| Category | Naming |
| Severity | Warning |
| Code fix | Yes (rename) |

## Cause

A type or member name does not begin with an upper-case letter.

## Rule description

Types and members follow the .NET PascalCase convention: classes, structs,
records, enums, delegates, methods, properties, events, and enum members all
begin with an upper-case letter.

```csharp
// 👎 reported
public class widget { public void doThing() { } }

// 👍 PascalCase
public class Widget { public void DoThing() { } }
```

Overrides and explicit interface implementations are **not** reported — their
names are dictated by the base type or interface. Interfaces are covered by
[SST1302](SST1302.md).

## How to fix violations

Rename the element to PascalCase. The code fix renames it across the solution.
