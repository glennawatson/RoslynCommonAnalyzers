# SST1303 — Constant names should be PascalCase

| Item | Value |
| --- | --- |
| Category | Naming |
| Severity | Warning |
| Code fix | Yes (rename) |

## Cause

A `const` field does not begin with an upper-case letter.

## Rule description

Constants follow the .NET PascalCase convention regardless of accessibility.

```csharp
// 👎 reported
private const int maxRetries = 3;

// 👍 PascalCase
private const int MaxRetries = 3;
```

The StyleSharp counterpart of StyleCop's `SA1303`.

## How to fix violations

Rename the constant to PascalCase. The code fix renames it across the solution.
