# SST1307 — Accessible fields should be PascalCase

| Item | Value |
| --- | --- |
| Category | Naming |
| Severity | Warning |
| Code fix | Yes (rename) |

## Cause

A `public`, `internal`, or `protected` field does not begin with an upper-case
letter.

## Rule description

Externally visible fields follow the .NET PascalCase convention. (Prefer
properties over public mutable fields, but when a field is exposed it should be
PascalCase.)

```csharp
// 👎 reported
public int itemCount;

// 👍 PascalCase
public int ItemCount;
```

The StyleSharp counterpart of StyleCop's `SA1307`.

## How to fix violations

Rename the field to PascalCase. The code fix renames it across the solution.
