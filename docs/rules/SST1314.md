# SST1314 — Type parameters should begin with T

| Item | Value |
| --- | --- |
| Category | Naming |
| Severity | Warning |
| Code fix | Yes (rename) |

## Cause

A generic type parameter does not begin with the capital letter `T`.

## Rule description

Type parameters begin with `T`, optionally followed by a descriptive PascalCase
name — `T`, `TKey`, `TValue`, `TResult`.

```csharp
// 👎 reported
public class Cache<Key, Value> { }

// 👍 T-prefixed
public class Cache<TKey, TValue> { }
```

## How to fix violations

Rename the type parameter to begin with `T`. The code fix renames it across the
solution.
