# SST1402 — File should contain a single type

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | No |

## Cause

A file declares more than one top-level type.

## Rule description

One top-level type per file makes types easy to locate. Partial declarations of the
same type count once, and nested types are ignored.

```csharp
// 👎 reported — two top-level types in one file
internal class Widget { }
internal class Gadget { }
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

Move each extra type into its own file named after the type.
