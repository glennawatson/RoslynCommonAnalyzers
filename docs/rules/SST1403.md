# SST1403 — File should contain a single namespace

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | No |

## Cause

A file declares more than one namespace.

## Rule description

A file declares a single namespace, whether block-scoped or file-scoped.

```csharp
// 👎 reported — two namespaces in one file
namespace Widgets { }
namespace Gadgets { }
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

Move each extra namespace into its own file.
