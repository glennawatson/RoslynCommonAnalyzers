# SST1405 — Debug.Assert should provide message text

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | No |

## Cause

A `System.Diagnostics.Debug.Assert` call passes no message.

## Rule description

A message explains the failed assumption to whoever trips the assert.

```csharp
// 👎 reported
Debug.Assert(index >= 0);

// 👍 with a message
Debug.Assert(index >= 0, "Index should never be negative here.");
```

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

Pass a message describing the assertion.
