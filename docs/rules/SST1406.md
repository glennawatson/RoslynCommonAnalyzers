# SST1406 — Debug.Fail should provide message text

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | No |

## Cause

A `System.Diagnostics.Debug.Fail` call passes an empty message.

## Rule description

A message explains the failure to whoever trips it.

```csharp
// 👎 reported
Debug.Fail("");

// 👍 with a message
Debug.Fail("Reached an unreachable switch arm.");
```

The StyleSharp counterpart of StyleCop's `SA1406`.

## How to fix violations

Pass a message describing the failure.
