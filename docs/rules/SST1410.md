# SST1410 — Remove delegate parenthesis when possible

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | Yes (remove parentheses) |

## Cause

An anonymous method declares an empty parameter list.

## Rule description

When an anonymous method takes no parameters the empty parameter list adds nothing
and can be omitted.

```csharp
// 👎 reported
Action a = delegate() { };

// 👍
Action a = delegate { };
```

## How to fix violations

The code fix removes the empty parameter list.
