# SST1411 — Attribute constructor should not use unnecessary parenthesis

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | Yes (remove parentheses) |

## Cause

An attribute with no arguments declares an empty argument list.

## Rule description

When an attribute passes no arguments the empty parentheses can be omitted.

```csharp
// 👎 reported
[Obsolete()]

// 👍
[Obsolete]
```

## How to fix violations

The code fix removes the empty argument list.
