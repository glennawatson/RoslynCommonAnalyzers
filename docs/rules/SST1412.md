# SST1412 — Store files as UTF-8 with a byte order mark

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning (disabled by default) |
| Code fix | No |

## Cause

A source file is not stored as UTF-8 with a byte order mark.

## Rule description

This rule and [SST1450](SST1450.md) are mutually exclusive — enable whichever byte-order-
mark policy your project standardises on. Both are off by default.

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

Save the file as UTF-8 with a byte order mark.
