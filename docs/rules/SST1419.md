# SST1419 — Remove redundant modifiers

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |

## Cause

A modifier has no effect in its declaration context.

```csharp
public partial class OnlyPart { }
```

## Rule description

The current implementation reports a `partial` declaration with only one source part and
a redundant `sealed` member modifier. Genuine multipart declarations and meaningful
sealed overrides are not reported.

## How to fix violations

Remove the reported modifier. The provided code fix does this automatically.
