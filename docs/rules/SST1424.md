# SST1424 — Fields that are never reassigned should be readonly

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | Yes |
| Default | Disabled (opt-in) |

## Cause

A private instance field is assigned only by its initializer or instance constructors.

## Rule description

`readonly` records the intended immutability and prevents later reassignment. Partial
types, static fields, volatile fields, closure writes, and writes outside constructors
are excluded.

## How to fix violations

Add the `readonly` modifier. The provided code fix does this automatically.
