# SST1435 — Empty namespace declarations should be removed

| Item | Value |
| --- | --- |
| Category | Maintainability |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |
| Analyzer | `EmptyCodeAnalyzer` |

## Cause

A namespace declaration contains no members.

```csharp
// 👎 reported
namespace Company.Product.Internal
{
}

// 👍 — delete the empty namespace
```

## Rule description

A block or file-scoped namespace declaration with no members is reported. It serves no purpose and can
be deleted. The check is purely syntactic.

## How to fix violations

Delete the empty namespace declaration. The provided code fix does this automatically.
