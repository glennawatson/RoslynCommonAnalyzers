# SST1454 - Validate composite format placeholders

**Category:** Maintainability
**Severity:** Warning
**Code fix:** None

A constant composite format string should only reference arguments that were
actually supplied. Out-of-range holes are runtime failures waiting in otherwise
compile-time-visible data.

The analyzer reports constant format strings whose numeric placeholders cannot
bind to the available argument list. It stays syntax-first and deliberately
ignores non-constant format strings.

## Example

```csharp
string Message(string name)
{
    return string.Format("Hello {1}", name); // SST1454
}
```

Use a valid placeholder or supply the missing argument.
