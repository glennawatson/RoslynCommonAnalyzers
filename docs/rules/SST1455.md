# SST1455 - Remove unused unsafe modifiers

**Category:** Maintainability
**Severity:** Warning
**Code fix:** None

An `unsafe` modifier should mark declarations that actually contain unsafe
syntax. If a declaration has no pointer, function-pointer, `fixed`, or unsafe
`sizeof` use, the modifier overstates the contract and makes the surrounding
code look riskier than it is.

## Example

```csharp
public unsafe int Count(ReadOnlySpan<int> values) // SST1455
{
    return values.Length;
}
```

Remove the modifier, or keep it only on the declaration that owns the unsafe
operation.
