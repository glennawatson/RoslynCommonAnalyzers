# SST1457 - Keep global suppression targets resolvable

**Category:** Maintainability
**Severity:** Warning
**Code fix:** None

An assembly-level `SuppressMessage` target should resolve to a declaration in the
current compilation. When the target no longer resolves, the suppression is stale
and should be fixed or removed.

The analyzer binds only assembly-level `SuppressMessage` attributes and uses
Roslyn's documentation-id resolver for target lookup.

## Example

```csharp
[assembly: SuppressMessage("Style", "SST1000", Justification = "Reviewed.", Scope = "member", Target = "M:C.Missing")] // SST1457
```
