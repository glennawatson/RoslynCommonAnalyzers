# SST1458 - Use declaration ids directly in global suppressions

**Category:** Maintainability
**Severity:** Warning
**Code fix:** None

Global `SuppressMessage` targets should use declaration documentation ids
directly. A legacy `~` prefix makes the target string a special case and should
be removed.

## Example

```csharp
[assembly: SuppressMessage("Style", "SST1000", Justification = "Reviewed.", Scope = "member", Target = "~M:C.M")] // SST1458
```

Use:

```csharp
[assembly: SuppressMessage("Style", "SST1000", Justification = "Reviewed.", Scope = "member", Target = "M:C.M")]
```
