# SST1459 - Remove non-grouping parentheses

**Category:** Maintainability
**Severity:** Warning
**Code fix:** None

Parentheses are useful when they clarify precedence. They are noise when they
wrap a standalone expression that is already isolated by its parent syntax, such
as a return value, argument, initializer, or assignment value.

The analyzer reports only simple operands in already-isolated contexts. It does
not try to solve general expression simplification.

## Example

```csharp
int M(int value) => (value); // SST1459
```

Use:

```csharp
int M(int value) => value;
```
