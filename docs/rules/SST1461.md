# SST1461 - Remove unread private parameters

**Category:** Maintainability
**Severity:** Warning
**Code fix:** None

A parameter on a private method or local function that is never read is dead
local API surface. Removing it keeps call sites and implementation intent in
sync.

The analyzer is scoped to private methods and local functions. Public,
overriding, virtual, abstract, partial, attributed, and explicit-interface
members are skipped because their signatures can be consumed outside the local
body.

## Example

```csharp
private int M(int value) => 1; // SST1461
```

Use:

```csharp
private int M() => 1;
```
