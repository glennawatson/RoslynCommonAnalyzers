# SST1465 - Collapse an else block that only wraps an if

**Category:** Maintainability
**Severity:** Warning
**Code fix:** Hoists the inner if into an `else if` chain (supports Fix All)

An `else` block whose only statement is an `if` statement adds a level of
nesting without changing which branch runs. Written as `else if`, the chain
stays flat and reads in the same order the conditions are tested.

The analyzer reports the `else` keyword when its block contains exactly one
statement and that statement is an `if` (with or without its own `else`
chain). A brace-free `else if`, a block with more than one statement, and a
block wrapping any other statement kind are not reported. The code fix keeps
the inner if's own `else` chain untouched and carries comments written inside
the removed braces onto the hoisted `if`.

## Example

```csharp
void M(int value)
{
    if (value > 0)
    {
        Positive();
    }
    else // SST1465
    {
        if (value < 0)
        {
            Negative();
        }
    }
}
```

Use:

```csharp
void M(int value)
{
    if (value > 0)
    {
        Positive();
    }
    else if (value < 0)
    {
        Negative();
    }
}
```
