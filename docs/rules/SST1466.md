# SST1466 - Remove case labels that share a section with default

**Category:** Maintainability
**Severity:** Warning
**Code fix:** Removes the redundant case label (supports Fix All)

A case label in the same switch section as the `default:` label selects the
body the default already provides. Listing the value adds nothing at runtime
and suggests the case receives special handling when it does not.

The analyzer reports each non-default label in a section that also carries the
default label. A switch containing any `goto case` or `goto default` statement
is skipped entirely, because the jump may target a label the rule would
otherwise remove.

## Example

```csharp
int M(int value)
{
    switch (value)
    {
        case 0:
            return 1;
        case 1: // SST1466
        default:
            return 0;
    }
}
```

Use:

```csharp
int M(int value)
{
    switch (value)
    {
        case 0:
            return 1;
        default:
            return 0;
    }
}
```
