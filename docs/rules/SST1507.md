# SST1507 — Code should not contain multiple blank lines in a row

| Item | Value |
| --- | --- |
| Category | Layout |
| Severity | Warning |
| Code fix | Yes (remove extra blank lines) |

## Cause

Two or more blank lines appear in a row.

## Rule description

A single blank line is enough to separate code. Blank lines inside a multi-line
string literal are part of the literal and are ignored.

```csharp
// 👎 reported
var a = 1;


var b = 2;

// 👍 single blank line
var a = 1;

var b = 2;
```

## How to fix violations

The code fix collapses the run of blank lines down to a single blank line.

## Performance

This rule looks at the file as a whole rather than a single declaration. It is still
evaluated once per file and the cost is negligible for normal source files — it is only
marginally heavier than the single-declaration rules.
