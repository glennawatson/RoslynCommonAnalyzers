# SST1509 — Opening brace should not be preceded by a blank line

| Item | Value |
| --- | --- |
| Category | Layout |
| Severity | Warning |
| Code fix | Yes (remove blank line) |

## Cause

A blank line immediately precedes an opening brace.

## Rule description

An opening brace follows its declaration directly, not after a blank line.

```csharp
// 👎 reported
private void M()

{
    Work();
}

// 👍 no blank line before the brace
private void M()
{
    Work();
}
```

## How to fix violations

The code fix removes the blank line(s) directly before the opening brace.
