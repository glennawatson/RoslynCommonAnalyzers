# SST1601 — Partial elements should be documented

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

An exposed partial type or partial method has no documentation comment.

## Rule description

Partial types and methods that are part of the public API carry XML documentation, just
like non-partial elements ([SST1600](SST1600.md) covers the non-partial case).

The StyleSharp counterpart of the analyzer's `the rule`.

## How to fix violations

Add a documentation comment to the partial element.
