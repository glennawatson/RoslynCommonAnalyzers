# SST1604 — Element documentation should have a summary

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A member has a documentation comment that does not contain a `<summary>`.

## Rule description

When a member is documented, the documentation should include a `<summary>`
(unless it uses `<inheritdoc>`).

The StyleSharp counterpart of StyleCop's `SA1604`.
