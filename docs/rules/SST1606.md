# SST1606 — Summary documentation should have text

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A `<summary>` element is empty.

## Rule description

A `<summary>` should contain descriptive text rather than being left blank.

The StyleSharp counterpart of StyleCop's `SA1606`.
