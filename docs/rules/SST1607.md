# SST1607 — Partial element summary should have text

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning (disabled by default) |
| Code fix | No |

## Cause

A documented partial element has an empty `<summary>`.

## Rule description

Off by default, matching StyleCop. Enable it to require summary text on documented
partial elements.

The StyleSharp counterpart of StyleCop's `SA1607`.

## How to fix violations

Add descriptive text to the `<summary>` element.
