# SST1610 — Property value documentation should have text

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning (disabled by default) |
| Code fix | No |

## Cause

A documented property has an empty `<value>` element.

## Rule description

Off by default. Enable it to require text in the `<value>` element.

## How to fix violations

Add descriptive text to the `<value>` element.
