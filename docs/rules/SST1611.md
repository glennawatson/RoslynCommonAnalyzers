# SST1611 — Parameters should be documented

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A documented member has a parameter with no matching `<param>` element.

## Rule description

```csharp
/// <summary>Configures the host.</summary>
/// <param name="host">The host name.</param>
public void Configure(string host, int port) { } // 👎 'port' is undocumented
```

The StyleSharp counterpart of the analyzer's `the rule`.
