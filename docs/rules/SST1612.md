# SST1612 — Parameter documentation should match the parameters

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A `<param>` element names a parameter the member does not have.

## Rule description

Every `<param>` refers to a real parameter; remove or rename stale ones. The StyleSharp counterpart of StyleCop's `SA1612`.
