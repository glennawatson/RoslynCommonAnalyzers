# SST1613 — Parameter documentation should declare the parameter name

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A `<param>` element has no `name` attribute.

## Rule description

Each `<param>` declares the parameter it documents via `name="..."`. The StyleSharp counterpart of the analyzer's `the rule`.
