# SST1615 — Return value should be documented

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A documented member with a non-void return type has no `<returns>` element.

## Rule description

```csharp
/// <summary>Gets the count.</summary>
/// <returns>The number of items.</returns>
public int GetCount() => 0;
```
