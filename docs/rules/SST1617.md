# SST1617 — Void return value should not be documented

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A member that returns `void` (or a `void`-returning delegate) has a `<returns>`
element.

## Rule description

A `void` member has nothing to return, so it should not carry a `<returns>`.

The StyleSharp counterpart of StyleCop's `SA1617`.
