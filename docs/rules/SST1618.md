# SST1618 — Generic type parameters should be documented

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A documented generic member has a type parameter with no matching `<typeparam>`
element.

## Rule description

```csharp
/// <summary>A typed cache.</summary>
/// <typeparam name="TKey">The key type.</typeparam>
/// <typeparam name="TValue">The value type.</typeparam>
public sealed class Cache<TKey, TValue> { }
```

The StyleSharp counterpart of the analyzer's `the rule`.
