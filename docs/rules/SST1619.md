# SST1619 — Partial generic type parameters should be documented

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning (disabled by default) |
| Code fix | No |

## Cause

A documented partial generic type has a type parameter without a matching `<typeparam>`
element.

## Rule description

Off by default. Enable it to require every generic type parameter of a
documented partial type to have a `<typeparam>` element.

## How to fix violations

Add a `<typeparam name="...">` element for each type parameter.
