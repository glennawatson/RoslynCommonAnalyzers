# SST1620 — Type parameter documentation should match the type parameters

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A `<typeparam>` element names a type parameter the member does not have.

## Rule description

Every `<typeparam>` refers to a real type parameter; remove or rename stale ones.
