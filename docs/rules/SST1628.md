# SST1628 — Documentation text should begin with a capital letter

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning (disabled by default) |
| Code fix | No |

## Cause

A documentation summary begins with a lower-case letter.

## Rule description

Off by default. Enable it to require summaries to read as proper
sentences beginning with a capital letter.

## How to fix violations

Capitalise the first letter of the summary.
