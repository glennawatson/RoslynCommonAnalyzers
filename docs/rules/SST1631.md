# SST1631 — Documentation text should be mostly letters

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning (disabled by default) |
| Code fix | No |

## Cause

Less than half of a documentation summary's characters are letters or whitespace.

## Rule description

Off by default. Enable it to flag summaries made up mostly of symbols
or punctuation rather than real prose.

## How to fix violations

Rewrite the summary as readable prose.
