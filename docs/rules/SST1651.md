# SST1651 — Documentation should not contain placeholder text

| Item | Value |
| --- | --- |
| Category | Documentation |
| Severity | Warning |
| Code fix | No |

## Cause

A documentation comment contains a generated `<placeholder>` element.

## Rule description

`<placeholder>` elements are inserted by tooling as a stub and are replaced with real
documentation before the code is committed.

```xml
<!-- 👎 reported -->
/// <summary><placeholder>Fill this in</placeholder></summary>

<!-- 👍 real documentation -->
/// <summary>Resets the buffer to its initial state.</summary>
```

## How to fix violations

Replace the `<placeholder>` element with real documentation.
