# SST2003 — Use ObjectDisposedException.ThrowIf

| Item | Value |
| --- | --- |
| Category | Modernization |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled when the runtime helper exists |

## Cause

An instance method performs a standard disposed check and throws
`ObjectDisposedException`.

## Rule description

.NET 8 provides `ObjectDisposedException.ThrowIf`, which expresses the guard directly.
The rule is runtime-gated, excludes static contexts, and avoids custom-message forms.

## How to fix violations

Replace the conditional throw with `ObjectDisposedException.ThrowIf(condition, this)`.
The provided code fix does this automatically.
