# SST2007 - Declare the checked type in the pattern

| Item | Value |
| --- | --- |
| Category | Modernization |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled |
| Analyzer | `PatternMatchingAnalyzer` |

## Cause

An `is` type check is followed immediately by a local initialized with a cast of the same value.

```csharp
if (value is string)
{
    var text = (string)value;
}
```

## Rule description

A declaration pattern combines the type test and local declaration. The analyzer reports only
locals, parameters, and fields so the suggested shape does not duplicate a property or method call.

## How to fix violations

Declare the local in the pattern. The provided code fix removes the redundant cast local.
