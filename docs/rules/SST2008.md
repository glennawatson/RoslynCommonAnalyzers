# SST2008 - Use is-not patterns

**Category:** Modernization
**Severity:** Warning
**Code fix:** None

A negated pattern test reads more directly as an `is not` pattern. The rewrite
also removes the extra grouping needed around the positive pattern expression.

Declaration patterns are skipped because the declared variable is not available
from a negative pattern in the same way.

## Example

```csharp
if (!(value is null)) // SST2008
{
}
```

Use:

```csharp
if (value is not null)
{
}
```
