# SST2102 - Let the span target carry stack allocation

A `stackalloc` initializer assigned to `Span<T>` or `ReadOnlySpan<T>` can be written as a collection
expression. This keeps the element list in the same compact form used by other C# 12 collection targets
while still requiring an explicit span target.

## Example

```csharp
Span<int> values = stackalloc int[] { 1, 2, 3 };
```

Use:

```csharp
Span<int> values = [1, 2, 3];
```

## Notes

The analyzer runs only for C# 12 or later and only when the referenced framework exposes collection
builder support. It does not report targetless `var` initializers or non-span targets.
