# SST2104 - Return builder contents directly

A short-lived builder local that is created, populated with `Add`, and immediately converted can be
returned as a collection expression. This removes the temporary builder plumbing from small methods and
keeps the returned elements visible in one place.

## Example

```csharp
var builder = MyCollection.CreateBuilder<int>();
builder.Add(1);
builder.Add(2);
return builder.ToImmutable();
```

Use:

```csharp
return [1, 2];
```

## Notes

The rule is intentionally narrow. It reports only contiguous builder sequences with simple `Add` calls
and a terminal materialization call, and the return target must be an array or a collection-builder-backed
type.
