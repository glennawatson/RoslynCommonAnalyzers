# SST2105 - Use the target collection expression directly

An inline array immediately materialized with LINQ can be written as a collection expression at the target
site. The target type already tells the compiler which collection shape is required.

## Example

```csharp
List<int> values = new[] { 1, 2, 3 }.ToList();
```

Use:

```csharp
List<int> values = [1, 2, 3];
```

## Notes

The analyzer checks that the materialization call resolves to `System.Linq.Enumerable.ToArray` or
`System.Linq.Enumerable.ToList`, and it requires an explicit target type so overload inference is not
reopened from a targetless initializer.
