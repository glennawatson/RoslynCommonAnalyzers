# SST2201 - Express return-only switch as a value

| Item | Value |
| --- | --- |
| Category | ModernSyntax |
| Severity | Warning |
| Code fix | Yes |
| Default | Enabled; C# 8 or later |
| Analyzer | `Sst2201PreferSwitchExpressionAnalyzer` |

## Cause

A switch statement returns or throws from every section and includes a `default` section.

```csharp
switch (value)
{
    case 0:
        return 1;
    default:
        return 2;
}
```

## Rule description

A return-only switch statement is expression-shaped. The analyzer reports simple switches whose
sections each contain one `return` or `throw` statement and one label.

## How to fix violations

Rewrite the switch as a return statement containing a switch expression. The provided code fix keeps
the existing arms in switch-expression form.
