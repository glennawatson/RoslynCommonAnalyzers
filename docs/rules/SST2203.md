# SST2203 - Index from the end directly

Array and string indexing from the end should use the C# index operator instead of repeating a
`Length - n` calculation inside the indexer.

## Example

```csharp
var last = values[values.Length - 1];
```

Use:

```csharp
var last = values[^1];
```

## Notes

The analyzer is limited to local and parameter receivers whose type is an array or `string`. It does
not report property receivers or collection `Count` patterns, avoiding repeated property/indexer
semantics and keeping the checked replacement allocation-free at runtime.
