# SST2204 - Slice strings with range syntax

String slices should use range syntax when the old call is a simple `Substring` over stable bounds.

## Example

```csharp
var slice = text.Substring(start, length);
```

Use:

```csharp
var slice = text[start..(start + length)];
```

## Notes

The analyzer only reports string receivers held in locals or parameters and only for literal, local,
or parameter bounds. It deliberately avoids property receivers and computed bounds whose repeated
evaluation could hide work or side effects.
