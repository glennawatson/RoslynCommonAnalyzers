# SST2205 - List every enum switch statement case

Enum switch statements should name every enum value unless they intentionally use a `default`
section.

## Example

```csharp
switch (mode)
{
    case Mode.First:
        break;
}
```

Use:

```csharp
switch (mode)
{
    case Mode.First:
        break;
    case Mode.Second:
        break;
}
```

## Notes

The analyzer checks enum symbols and existing case labels directly. The code fix adds explicit
sections for missing values and leaves switches with `default` alone.
