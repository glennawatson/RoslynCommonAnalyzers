# SST2206 - List every enum switch expression arm

Enum switch expressions should name every enum value unless they intentionally use a discard arm.

## Example

```csharp
var result = mode switch
{
    Mode.First => 1
};
```

Use:

```csharp
var result = mode switch
{
    Mode.First => 1,
    Mode.Second => throw new System.NotImplementedException()
};
```

## Notes

The analyzer works from the enum symbol and constant patterns, not text matching. The code fix adds
throwing arms for missing values so the new branches stay visible and must be completed deliberately.
