# SST2207 - Fold null guard into returned value

A null guard that throws and immediately returns the guarded value can keep the throw beside the
returned expression.

## Example

```csharp
if (value is null)
{
    throw new ArgumentNullException(nameof(value));
}

return value;
```

Use:

```csharp
return value ?? throw new ArgumentNullException(nameof(value));
```

## Notes

The analyzer only reports a single guard followed by `return value;` for the same symbol. It skips
guards with comments, extra work, `else` branches, or different returned expressions.
