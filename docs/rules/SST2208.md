# SST2208 - Declare out variables at the call site

An `out` temporary declared solely for the next statement should be declared where the `out` value is
first written.

## Example

```csharp
int value;
if (int.TryParse(text, out value))
{
    return value;
}
```

Use:

```csharp
if (int.TryParse(text, out var value))
{
    return value;
}
```

## Notes

The analyzer only reports one uninitialized local consumed by one `out` argument in the immediately
following statement. Declarations with comments, initializers, or multiple matching writes are left
unchanged.
