# SST2209 - Remove a no-op null-forgiving operator

A null-forgiving operator should be removed when it is applied to a value that cannot affect nullable
flow.

## Example

```csharp
var value = 1;
return value!;
```

Use:

```csharp
var value = 1;
return value;
```

## Notes

The analyzer reports non-nullable value-type operands and deliberately avoids nullable reference
symbols where the suppression may be documenting a real contract boundary.
