# SST2213 - Remove no-op discard designation

A typed pattern does not need an explicit discard name. The type alone carries the same match and avoids
making the reader scan a binding that is never used.

## Example

```csharp
if (value is int _)
{
    return true;
}
```

Use:

```csharp
if (value is int)
{
    return true;
}
```

## Notes

The analyzer skips broad `var _` patterns, because removing the designation there would not leave a
useful type test behind.
