# SST2215 - Swap locals with tuple assignment

A temporary local used only to swap two locals can be replaced by tuple assignment. The shorter form
keeps the operation as one expression and removes the throwaway variable.

## Example

```csharp
var temp = left;
left = right;
right = temp;
```

Use:

```csharp
(left, right) = (right, left);
```

## Notes

The analyzer only reports simple local-or-parameter swaps. Property, indexer, and field swaps are not
reported because tuple assignment can change receiver or setter evaluation in those shapes.
