# SST2216 - Let tuple element names be inferred

When a tuple element repeats the exact name C# would infer from the expression, the explicit name can be
omitted.

## Example

```csharp
return (name: name, Age: person.Age);
```

Use:

```csharp
return (name, person.Age);
```

## Notes

The analyzer only removes names that match the compiler-inferred tuple element name. Deliberate aliases
remain unchanged.
