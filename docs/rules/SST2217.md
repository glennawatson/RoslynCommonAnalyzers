# SST2217 - Combine hash inputs with System.HashCode

A simple multiplier-based `GetHashCode` expression can use `System.HashCode.Combine` when the target
framework exposes it. The helper keeps the list of hash inputs visible and avoids hand-maintained
mixing code.

## Example

```csharp
public override int GetHashCode() => (Id.GetHashCode() * 397) ^ Age.GetHashCode();
```

Use:

```csharp
public override int GetHashCode() => System.HashCode.Combine(Id, Age);
```

## Notes

The analyzer checks for `System.HashCode.Combine` once per compilation before registering this rule. It
also keeps the first implementation deliberately narrow: only value-type receivers inside
`GetHashCode()` are reported, so replacing `x.GetHashCode()` cannot change null behavior.
