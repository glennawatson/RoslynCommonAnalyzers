# SST2219 - Keep simple accessors expression-shaped

A property accessor whose body is just a return or assignment can use an expression body. This removes
brace noise while preserving the accessor boundary.

## Example

```csharp
get { return _value; }
set { _value = value; }
```

Use:

```csharp
get => _value;
set => _value = value;
```

## Notes

Only single-statement `get`, `set`, and `init` accessors are reported. Accessors that already have
expression bodies or contain local setup stay unchanged.
