# SST2220 - Let interpolation format the value

An interpolation hole that only calls `ToString()` can let the interpolation syntax carry the value
directly. Literal format strings are kept as interpolation format text, which makes the formatted value
easier to scan.

## Example

```csharp
$"Value: {value.ToString("X")}"
```

Use:

```csharp
$"Value: {value:X}"
```

## Notes

Only direct `ToString()` calls and `ToString("literal-format")` calls are reported. Culture-aware or
computed format calls are left alone because moving them would change the call shape.
