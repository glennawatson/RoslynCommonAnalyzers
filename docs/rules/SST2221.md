# SST2221 - Mark ignored expression values

An expression statement that returns a value can assign that value to `_` when the result is intentionally
discarded. This makes the ignored result explicit without introducing a named local.

## Example

```csharp
Compute();
```

Use:

```csharp
_ = Compute();
```

## Notes

This rule is opt-in. It is useful in codebases that want ignored return values to be visible, but it can
be noisy around fluent APIs whose returned receiver is normally ignored.
