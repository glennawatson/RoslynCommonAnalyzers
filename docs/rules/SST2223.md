# SST2223 - Assign fallback values with ??=

A null check whose only job is to assign a fallback value can use `??=`. The fallback stays beside the
target being updated, and the control-flow block disappears.

## Example

```csharp
if (value is null)
{
    value = fallback;
}
```

Use:

```csharp
value ??= fallback;
```

## Notes

The rewrite is only offered for C# versions that support coalescing assignment. The target must be safe
to read for the null check and the assignment, so member chains with unknown side effects are skipped.
