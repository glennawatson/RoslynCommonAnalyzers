# SST2224 - Use tuple syntax for local value bundles

A small anonymous object used as a local value bundle can be written as a tuple when all element names
can be represented directly in tuple syntax. This keeps the value-shape lightweight at the call site.

## Example

```csharp
return new { id, Label = name };
```

Use:

```csharp
return (id, Label: name);
```

## Notes

This rule is opt-in because anonymous objects and tuples have different runtime types, equality behavior,
and member names in reflection. It is intended for codebases that already treat these shapes as local
style-only bundles.
