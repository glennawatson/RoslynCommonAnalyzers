# SST2225 - Show foreach element casts at the source

A `foreach` loop over a strongly typed source can hide a runtime element cast in the loop header. Moving
the cast to the source expression makes the conversion visible before the body starts.

## Example

```csharp
foreach (string value in values)
{
}
```

Use:

```csharp
foreach (string value in System.Linq.Enumerable.Cast<string>(values))
{
}
```

## Notes

The analyzer uses Roslyn's foreach binding information and only reports explicit element conversions.
Legacy non-generic enumeration shapes are skipped so the rule does not turn every old `IEnumerable`
loop into a LINQ call.
