# SST2226 - Show the inner explicit conversion

A cast can cause another explicit conversion before the final target conversion. Writing the inner cast
out makes the conversion chain visible to the reader and to future edits.

## Example

```csharp
Derived value = (Derived)source;
```

Use:

```csharp
Derived value = (Derived)(Base)source;
```

## Notes

The analyzer only reports casts where Roslyn exposes an implicit conversion operation that still carries
an explicit conversion internally. Simple direct casts are not reported.
