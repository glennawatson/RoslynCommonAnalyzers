# SST2227 - Fold null fallback into the assignment

When a value is assigned and then immediately checked for null, the fallback or throw expression can sit
beside the assigned expression. This keeps the null behavior attached to the value it protects.

## Example

```csharp
string value = input;
if (value == null)
{
    throw new InvalidOperationException();
}
```

Use:

```csharp
string value = input ?? throw new InvalidOperationException();
```

## Notes

The analyzer supports immediate throw and fallback-assignment bodies. It only reports reference-like
values and keeps side-effecting assignment targets out of the rewrite.
