# SST2228 - Prefer a local function for local call sites

Delegate locals initialized with a lambda allocate delegate-shaped state when the name is only used as a
direct call target in the same block. A local function keeps the helper as method-shaped code and makes the
call site read like a normal local operation.

## Example

```csharp
Func<int, int> add = value => value + 1;
return add(input);
```

Use:

```csharp
int add(int value) => value + 1;
return add(input);
```

## Notes

The analyzer only reports delegate locals whose later references are direct invocations. If the delegate
value is stored, passed, returned, or otherwise used as a value, the rule stays silent.
