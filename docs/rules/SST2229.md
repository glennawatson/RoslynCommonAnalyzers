# SST2229 - Carry LINQ predicates at the terminal call

A `Where` call followed immediately by a terminal LINQ operator with a predicate overload creates an
extra iterator layer. Moving the predicate into the terminal call keeps the same intent with less sequence
machinery.

## Example

```csharp
return values.Where(value => value > 0).Any();
```

Use:

```csharp
return values.Any(value => value > 0);
```

## Notes

The analyzer uses syntax-first matching and then verifies both calls resolve to `System.Linq.Enumerable`.
It currently targets terminal operators that have matching predicate overloads.
