# SST2231 - State null checks directly

A broad `object` pattern over a reference value only proves nullness. A direct null pattern says that
without implying that the runtime type matters.

## Example

```csharp
return value is object;
```

Use:

```csharp
return value is not null;
```

## Notes

The analyzer requires C# 9 pattern syntax and skips value-type expressions so it does not introduce
boxing-shaped checks.
