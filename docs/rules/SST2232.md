# SST2232 - Omit generic arguments inside nameof

`nameof` returns the generic type name without considering concrete type arguments. When the parser
supports omitted generic arguments in `nameof`, the concrete arguments can be removed.

## Example

```csharp
return nameof(Dictionary<string, int>);
```

Use:

```csharp
return nameof(Dictionary<,>);
```

## Notes

The analyzer is language-version gated and only reports when C# 14 or later syntax is available.
