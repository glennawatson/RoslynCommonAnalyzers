# SST2235 - Make capture-free local functions static

**Category:** ModernSyntax
**Severity:** Warning
**Code fix:** None

A local function that does not capture locals, parameters, or instance state can
be declared `static`. That makes the no-capture contract explicit and prevents a
later edit from accidentally depending on outer state.

Recursive local functions are allowed when the only self-reference is the
function itself.

## Example

```csharp
int Run(int value)
{
    int Twice(int input) => input * 2; // SST2235
    return Twice(value);
}
```

Use:

```csharp
int Run(int value)
{
    static int Twice(int input) => input * 2;
    return Twice(value);
}
```
