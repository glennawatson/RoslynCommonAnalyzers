# SST2236 - Use using declarations for tail-position using blocks

**Category:** ModernSyntax
**Severity:** Warning
**Code fix:** None

A `using` block that is already the final statement in its containing block can
be written as a using declaration without changing when disposal happens. This
removes one indentation level while preserving the lifetime.

The analyzer reports only tail-position blocks so the rewrite does not extend
the disposable value's lifetime past following statements.

## Example

```csharp
void Write(Stream stream)
{
    using (var writer = new StreamWriter(stream)) // SST2236
    {
        writer.Write("done");
    }
}
```

Use:

```csharp
void Write(Stream stream)
{
    using var writer = new StreamWriter(stream);
    writer.Write("done");
}
```
