# SST2237 - Use file-scoped namespaces

**Category:** ModernSyntax
**Severity:** Warning
**Code fix:** None

A file with one block-scoped namespace and no sibling declarations can use a
file-scoped namespace declaration. The file-scoped form keeps the namespace at
the top of the file and removes one indentation level from every contained
member.

## Example

```csharp
namespace App.Services // SST2237
{
    internal sealed class Worker;
}
```

Use:

```csharp
namespace App.Services;

internal sealed class Worker;
```
