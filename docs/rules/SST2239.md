# SST2239 - Use method groups for forwarding lambdas

**Category:** ModernSyntax
**Severity:** Warning
**Code fix:** None

A lambda that only forwards its parameters to one method call can be written as a
method group. The method-group form removes a redundant parameter list and keeps
the target method as the visible operation.

The analyzer reports only straightforward forwarding lambdas where the argument
order matches the lambda parameter order.

## Example

```csharp
Func<int, int> transform = value => Normalize(value); // SST2239
```

Use:

```csharp
Func<int, int> transform = Normalize;
```
