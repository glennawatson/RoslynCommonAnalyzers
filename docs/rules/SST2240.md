# SST2240 - Use conditional delegate invocation

**Category:** ModernSyntax
**Severity:** Warning
**Code fix:** None

A delegate checked for null and then invoked can use conditional invocation.
`?.Invoke(...)` evaluates the delegate target once and expresses the null guard
at the call site.

## Example

```csharp
if (Changed != null) // SST2240
{
    Changed(this, EventArgs.Empty);
}
```

Use:

```csharp
Changed?.Invoke(this, EventArgs.Empty);
```
