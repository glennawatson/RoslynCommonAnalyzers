# SST2241 - Move simple constructor storage to the type declaration

**Category:** ModernSyntax
**Severity:** Warning
**Code fix:** None

A class or struct constructor whose body only copies each parameter into an
instance field or property can use primary-constructor parameters. The storage
wiring then sits at the type boundary rather than in a boilerplate constructor
body.

The analyzer requires every parameter to be stored exactly once and skips
records, types that already have a primary constructor, constructor initializers
with arguments, and constructors with extra work.

## Example

```csharp
public sealed class C
{
    private readonly int _value;

    public C(int value) // SST2241
    {
        _value = value;
    }
}
```

Use a primary constructor and move the storage initializer to the member.
