# SST2242 - Complete enum switch statement mappings

**Category:** ModernSyntax
**Severity:** Warning
**Code fix:** None

A switch statement that maps enum values should either name every enum member or
include an intentional catch-all section. Missing values make enum growth and
forgotten cases hard to audit.

This is narrower than broad switch-statement coverage: a `default` section is
treated as intentional, and empty fall-through-shaped sections are skipped.

## Example

```csharp
switch (color) // SST2242 when Color.Blue is missing
{
    case Color.Red:
        return 1;
}
```

Add the missing enum cases or an intentional `default` section.
