# SST2276 — Keep a single-statement constructor expression-shaped

| Item | Value |
| --- | --- |
| Category | ModernSyntax |
| Severity | Warning |
| Code fix | Yes (supports Fix All) |
| Default | Disabled (opt-in) |

## Cause

A constructor's block body is a single call.

```csharp
public Client(HttpClient http)
{
    Configure(http);
}
```

## Rule description

A constructor whose body runs one statement can be written with an expression body.

```csharp
public Client(HttpClient http) => Configure(http);
```

Whether a constructor reads better on one line is more contested than for a plain method, so the rule ships
**disabled**. Enable it in `.editorconfig` for a codebase that prefers it.

The rewrite is pure syntax and never changes what the constructor does. Because the collapsed body no longer
lives inside a block, the fix also settles any single-line-block warning (SST1501/SST1502) on the same
constructor — the expression-body form supersedes it.

## What is not reported

**A constructor with a `: this(...)` or `: base(...)` initializer.** The rule stays out of any constructor
that chains to another, so nothing is dropped or reordered in the rewrite.

**A block with more than one statement.**

**A single-statement body that carries a comment**, which the collapse would lose.

## How to fix violations

The fix replaces the block body with its single expression, pulls the arrow up beside the parameter list,
and keeps the constructor's trailing trivia.

```csharp
// before
public Client(HttpClient http)
{
    Configure(http);
}

// after
public Client(HttpClient http) => Configure(http);
```

## Enabling the rule

```ini
[*.cs]
dotnet_diagnostic.SST2276.severity = warning
```

## Language versions

Expression-bodied constructors are available from C# 7, so the rule stays silent below that language
version.

## Performance

The rule is a pure-syntax check on constructor nodes — no initializer, and a body of exactly one expression
statement — and never consults the semantic model. It is disabled by default, so a project that has not
enabled it pays nothing at all.
