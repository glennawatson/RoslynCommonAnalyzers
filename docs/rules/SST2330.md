# SST2330 — Write a combined flags value as the members it combines

| Item | Value |
| --- | --- |
| Category | Design |
| Severity | Info |
| Code fix | Yes (rewrite the literal as `A | B | C`) |
| Default | Enabled |

## Cause

On a `[Flags]` enum, a member is assigned a bare number that equals the bitwise-OR of two or more of the
enum's own single-bit members.

```csharp
using System;

[Flags]
public enum Access
{
    None = 0,
    Read = 1,
    Write = 2,
    Execute = 4,
    All = 7,      // SST2330: 7 == Read | Write | Execute
}
```

## Rule description

A member written as a bare number hides its own meaning. The reader has to add the bits back up to see that
`7` is `Read | Write | Execute`, and — worse — the moment one of the underlying members is renumbered, the
literal is silently wrong while still compiling. Written as `Read | Write | Execute`, the value says what it
is and stays correct when the members it names change.

The rule resolves the members' constant values and only fires when the literal decomposes **exactly** into
two or more of the enum's single-bit members: every bit of the literal owned by a declared bit, and nothing
left over.

## What is reported

- A `[Flags]` enum member whose initializer is a numeric literal equal to an OR of two or more of the enum's
  single-bit members (`All = 7`, `ReadWrite = 3`).

## What is not reported

- A value already written as the OR of its members (`All = Read | Write | Execute`) — it is already saying
  what it means.
- A single-bit member: that is a flag definition, not a combination.
- A literal carrying a bit no member declares (`Mixed = 5` when nothing owns bit 4): it is not a clean
  combination of the enum's flags, so the rule stays silent rather than guess.
- An enum without the `[Flags]` attribute.

## How to fix violations

The code fix rewrites the literal into the OR of the members it combines:

```diff
 [Flags]
 public enum Access
 {
     None = 0,
     Read = 1,
     Write = 2,
     Execute = 4,
-    All = 7,
+    All = Read | Write | Execute,
 }
```

## Performance

The clean path is a type-kind and attribute check, so only a flags enum with at least two single-bit members
pays for the decomposition scan.
