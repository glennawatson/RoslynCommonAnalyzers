# SST2334 — Give a public type a debugger-display attribute

| Item | Value |
| --- | --- |
| Category | Design |
| Severity | Warning |
| Code fix | Yes (add a `[DebuggerDisplay]` skeleton) |
| Default | Disabled (opt-in) |

## Cause

A publicly visible class, struct, or record carries no `[System.Diagnostics.DebuggerDisplay]` attribute.

```csharp
public class Money   // SST2334
{
    public int Amount { get; set; }
}
```

## Rule description

In the debugger, a type with no display attribute shows as its bare type name and nothing else, so a watch
window or a collection of instances is opaque until each one is expanded. A `[DebuggerDisplay]` string surfaces
the one or two members that identify an instance at a glance, which pays for itself the first time you scan a
list of them.

This is an opinionated, heavy nudge that would otherwise fire on nearly every public type, so it is **disabled
by default** and opt-in through `.editorconfig`. The rule resolves the attribute type in the compilation and
stays silent when it is absent, so the fix it offers always has something to add.

## Opting in

```ini
[*.cs]
dotnet_diagnostic.SST2334.severity = warning
```

## What is reported

- An externally visible class or struct (including a record) with no `[DebuggerDisplay]` attribute.

## What is not reported

- A type that already carries the attribute.
- An `internal` or otherwise non-externally-visible type.
- A `static` class, which has no instances to display.

## How to fix violations

The code fix adds a `[DebuggerDisplay]` skeleton, naming the type's first public property when it has one and
otherwise falling back to `ToString()`:

```diff
+[System.Diagnostics.DebuggerDisplay("{Amount}")]
 public class Money
 {
     public int Amount { get; set; }
 }
```

The generated string is a starting point — refine it to the members that best identify an instance.

## Performance

The clean path is an attribute scan on a type that is already externally visible; a project whose framework has
no debugger-display attribute never registers the rule at all.
